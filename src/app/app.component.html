HEADS UP:

The following scenario was not working in Linux Ubuntu 20.04. It have worked on Mac OS.

According to Flex Layout Responsive API docs, lg applies the rule:

screen and (min-width: 1280px) and (max-width: 1919px)'

And xl applies:

'screen and (min-width: 1920px) and (max-width: 5000px)'

Knowing that, change your screen resolution and check the result:

  • to 1280px x 720px (HD size) and check if this div is blue. (must apply ngClass.lg with class class-lg)
  • to 1920px x 1080px (Full HD size) and check if this div is red. (must apply ngClass.xl with class class-xl)
